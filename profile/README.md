<h2>Purpose of StickyPad</h2>
<p>StickyPad is a small, lightweight application for Windows aimed at letting users place virtual sticky notes on the desktop. It serves as a digital replacement for paper sticky notes - enabling you to jot down reminders, to-do lists, quick notes or ideas and have them visible whenever you need. Because notes remain on the desktop (or hidden in tray) until dismissed, StickyPad helps users manage tasks, reminders and quick information without opening a full-fledged note-taking application.</p>

<h2>Main Features</h2>
<ul>
  <li>Create multiple sticky notes and place them anywhere on the desktop.</li>
  <li>Customize note appearance - color, font, text size.</li>
  <li>Set transparency (make note translucent) so notes can float above other windows without fully obstructing content.</li>
  <li>Option to "pin" notes - keep them always on top of other windows.</li>
  <li>Supports reminders / basic scheduling - you can set a reminder for a note.</li>
  <li>Notes are saved automatically; you can back them up or move them to another computer by copying the note files.
  <li>Runs on many versions of Windows - works even on older systems.</li>
  <li>Minimal system requirements; small size; lightweight and unobtrusive.</li>
</ul>

<h2>Unique Advantages</h2>
<p>StickyPad stands out by offering a very lightweight, simple and portable solution for desktop notes. Unlike heavy note-taking tools or overly complex applications, StickyPad delivers straightforward sticky-note functionality with minimal resource consumption. Its ability to make notes translucent or keep them always on top makes it flexible for users who want quick access to reminders without interfering with their workspace. Because note files are stored locally and automatically saved, users can backup or migrate their notes simply by copying files.</p>

<h2>Functional Modules & How It Works</h2>
<ul>
  <li><strong>Note creation module</strong> - lets you create a new note (e.g. via shortcut) and type freely.</li>
  <li><strong>Display & positioning engine</strong> - you can drag notes anywhere on desktop; set their z-order (always on top), transparency, color and font.</li>
  <li><strong>Reminder & timestamp module</strong> - allows adding reminders and automatically inserting date/time in notes.</li>
  <li><strong>Persistence & storage</strong> - notes are saved into local storage (AppData) and restored automatically at startup; notes can be backed up or migrated by copying files.</li>
  <li><strong>User interface & tray integration</strong> - StickyPad resides in system tray and can be accessed quickly; right-click menu provides options like create note, show all notes, shut down.</li>
</ul>

<h2>Supported Platforms & Technical Requirements</h2>
<p>StickyPad is designed for Microsoft Windows. It supports a wide range of Windows versions - from older systems (Windows 95 / 98 / ME / NT) up to modern Windows editions. The program is very small (installer ≈ 770 KB) and does not demand significant system resources, making it suitable even for older or low-end machines.</p>

<h2>Limitations and Considerations</h2>
<p>StickyPad is a simple desktop note utility - it does not offer advanced features found in full-feature note apps: no cloud synchronization, no advanced formatting, no collaboration, no encryption or advanced organization beyond basic notes. Notes are stored locally, so if you lose access to the PC (drive failure, reinstall), you must back up note files manually to avoid data loss. Also, because the program was last updated many years ago and the developer states no further updates are planned, it may not receive new features or official support.</p>
